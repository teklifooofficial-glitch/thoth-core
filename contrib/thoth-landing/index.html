Thoth (TTH) — Layer 1 (development)

Thoth is an open-source Scrypt proof-of-work blockchain derived from the Litecoin Core lineage. Consensus v2 (near-genesis BIPs, Segwit at block 144, MWEB disabled) is live on a small operator network. A public testnet soak started 2026-05-30. This is experimental software — not production-ready.

Status

  • Phase: 1 — public testnet & infrastructure
  • Mainnet: early network (~height 1, two known nodes)
  • Testnet soak: started 2026-05-30 (target end ~2026-06-29)
  • Exchange listed: No — TTH has no official market
